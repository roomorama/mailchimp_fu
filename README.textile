h1. mailchimp_fu

Mailchimp_fu is a collection of modules to help with the the integration
of MailChimp in your rails application. These modules and helpers are the
following:

* acts_as_mailchimp_subscriber # Automatic subscription and update of model instances to mailchimp lists
* MailChimp class # Direct ruby function access to the MailChimp API

h2. acts_as_mailchimp_subscriber

h3. Examples

<code>
class User < ActiveRecord::Base

	acts_as_mailchimp_subscriber :all_users
	acts_as_mailchimp_subscriber :unconfirmed, :if => Proc.new {|user| !user.confirmed?}

end
</code>

h2. MailChimp

MailChimp is a wrapper class around the MailChimp API. It will give you direct ruby programmatic access to most
of the MailChimp API functions.
